# 🚚 Safe Delivery | Entrega Segura

**Safe Delivery** é um simulador de condução urbana em perspectiva top-down desenvolvido para desafiar a prudência do motorista. Em vez de correr contra o relógio, seu objetivo é manter a integridade de sua carga em uma cidade viva e dinâmica.

## 🕹️ Como Jogar

1.  **Objetivo**: Entregue o pacote no ponto de destino sem destruir a carga ou causar acidentes fatais.
2.  **Controles**:
    *   `W` / `S`: Acelerar / Reverter
    *   `A` / `D`: Virar
    *   `ESPAÇO`: Freio de Mão
3.  **Regras**:
    *   Respeite os semáforos (Sinal Vermelho = Infração).
    *   Evite colisões (Dano à carga).
    *   Cuidado com pedestres (Falha Crítica).
    *   Reduza a velocidade na chuva para evitar derrapagens.

## 🛠️ Tecnologias Utilizadas

*   **HTML5 Canvas**: Renderização de alta performance.
*   **Vanilla JavaScript**: Motores de física, IA e lógica de jogo.
*   **Modern CSS**: Interface responsiva com Glassmorphism.

## 🚀 Deployment no GitHub Pages

Este projeto foi estruturado para rodar diretamente via GitHub Pages. Para publicar:

1.  Crie um repositório no GitHub.
2.  Faça o upload dos arquivos (`index.html`, `style.css`, `script.js`).
3.  Vá em **Settings > Pages**.
4.  Selecione a branch `main` e a pasta `/ (root)`.
5.  Salve e pronto!
